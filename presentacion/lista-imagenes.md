## Resumen de Imágenes Necesarias

**Total**: 28 imágenes

**Por tipo**:
- Capturas de pantalla de la aplicación: ~10 imágenes
- Diagramas y esquemas: ~15 imágenes
- Logos y gráficos: ~3 imágenes

**Prioridad**:
- **Alta**: Imágenes 01, 07, 10, 11, 12, 17, 20, 21 (capturas de la aplicación)
- **Media**: Imágenes 02, 04, 13, 14, 15 (diagramas técnicos)
- **Baja**: Imágenes 19, 22, 27 (pueden ser más simples o opcionales)

## Herramientas Recomendadas para Crear Diagramas

1. **Draw.io** (https://app.diagrams.net/) - Gratis, online, fácil de usar
2. **Lucidchart** - Profesional, requiere cuenta
3. **Canva** - Fácil, plantillas disponibles
4. **PowerPoint** - Si ya lo tienes instalado
5. **GIMP/Photoshop** - Para edición avanzada de imágenes

## Consejos para Capturas de Pantalla

1. **Resolución**: Usa la resolución más alta posible
2. **Formato**: PNG es preferible para capturas, mantiene mejor calidad
3. **Limpieza**: Asegúrate de que no haya ventanas o elementos distractores
4. **Consistencia**: Mantén el mismo tamaño de ventana para capturas similares
5. **Edición**: Puedes usar herramientas básicas para recortar o ajustar brillo/contraste

## Notas Importantes

- Todas las imágenes deben tener nombres exactos como se especifica
- Las extensiones pueden ser .png, .jpg, o .pdf (PNG recomendado)
- Si alguna imagen no está disponible, puedes comentar esa línea en el archivo .tex
- Las imágenes deben ser de alta calidad para una presentación profesional
- Considera el tamaño de archivo: optimiza imágenes grandes antes de incluirlas
